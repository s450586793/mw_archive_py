# 3D打印本地模型管理工具

一个面向 3D 打印场景的本地模型库工具，支持 MakerWorld 模型归档、本地 3MF 导入解析、模型浏览筛选、附件管理，以及 Telegram / 飞书通知。

## 功能

- 归档 MakerWorld 模型到本地
- 导入和解析本地 `3MF` 文件
- 浏览、搜索、筛选和管理模型库
- 支持模型详情页视频和附件展示
- 支持移动端访问
- 支持 Telegram / 飞书通知
