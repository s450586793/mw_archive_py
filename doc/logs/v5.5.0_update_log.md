# v5.5.0 更新日志

> 更新日期: 2026-03-09

## 更新概述
本次版本聚焦在线归档模型维护体验，重点补齐模型详情页在线编辑能力，并同步修复 `BUG-20260304-001`。现在可以直接在在线模型详情页修改元数据、编辑富文本简介，并在需要时快速恢复最近一次备份。

## 当前版本同步说明
- 当前版本：`v5.5.0`
- 更新说明：[`doc/logs/v5.5.0_update_log.md`](./v5.5.0_update_log.md)
- 本次重点：
  - 修复 `BUG-20260304-001`；
  - 新增在线模型详情页编辑功能；
  - 支持归档模型元数据自动备份与最近备份恢复。

## 详细变更

### 新增功能
- 新增在线模型详情页编辑入口：
  - 支持在详情页直接打开“编辑归档”弹窗
  - 支持编辑标题、标签、分类、版本备注
  - 支持简介富文本编辑
- 新增归档元数据备份与恢复能力：
  - 保存编辑内容前自动备份 `meta.json`
  - 支持“恢复最近备份”快速回滚最近一次修改

### 优化改进
- 调整详情页编辑按钮位置，使其与信息区操作样式保持一致
- 最近备份时间统一格式化为可读日期时间
- 编辑弹窗增强设计图管理能力：
  - 支持保留或删除已有设计图
  - 支持追加上传新设计图
  - 支持重新指定封面图

### Bug 修复
- 修复 `BUG-20260304-001`

## 兼容性与注意事项
- 本次不涉及数据结构破坏性变更，现有归档模型可直接升级使用
- 备份恢复当前以“最近一次备份”为入口，不提供多版本可视化比较
- 简介富文本编辑当前为轻量工具栏模式，复杂样式建议先整理后再粘贴

## 涉及文件
- `app/server.py` - 在线编辑 API、元数据备份与恢复接口
- `app/templates/model.html` - 详情页编辑入口与编辑弹窗结构
- `app/static/js/model.js` - 在线编辑交互、富文本简介、最近备份显示
- `app/static/css/model.css` - 详情页编辑入口与弹窗样式
- `doc/plan/REQ-20260304-004.md` - 在线编辑需求实现记录
- `README.md` - 当前版本区块同步为 `v5.5.0`
