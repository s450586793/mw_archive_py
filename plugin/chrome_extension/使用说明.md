# MakerWorld Chrome 插件使用说明

## 目录
- 插件目录：`plugin/chrome_extension/mw_quick_archive_ext`

## 安装
1. 打开 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 复制mw_quick_archive_ext.zip到自己目录,解压
4. 选择解压目录：`mw_quick_archive_ext`

## 功能
- 配置后端 API 地址（默认：`http://127.0.0.1:8000`）
- 配置 API Token（公网部署时必填）
- 测试连接（请求 `GET /api/config`）
- 一键归档当前模型（请求 `POST /api/archive`）
- 同步 Cookie（自动读取）
- 手动设置 Cookie（扩展选项页）
- 打开本地模型主页（打开配置的 API 地址）
- 归档成功后可直接「打开本地模型」

## 使用流程
1. 登录网页控制台，在“安全设置”页面生成一个 API Token
2. 点击扩展图标，先「保存地址」
3. 打开扩展程序选项页，填写 API Token
4. 回到扩展弹窗执行「测试连接」
5. 打开 MakerWorld 模型页面（`https://makerworld.com.cn/zh/models/*`）
6. 点击「归档当前模型」
7. 若需要，点击「打开本地模型」查看归档页

## 手动 Cookie 设置
1. 扩展页点击「扩展程序详情」->「扩展程序选项」
2. 在「API Token」输入控制台生成的 Bearer Token
3. 在「手动 Cookie」输入完整 Cookie 字符串
4. 点击「保存并同步手动 Cookie」
