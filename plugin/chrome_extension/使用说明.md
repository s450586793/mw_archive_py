# Chrome 插件使用说明

## 版本更新
- `v1.0.2`（2026-04-01）
  - 模型页新增“归档状态标记”（已归档绿色 / 未归档灰色）
  - 归档标记默认开启，样式与关注按钮统一
  - 弹窗新增“标记开/标记关”按钮，位置在「保存地址 / 测试连接」同一行最右侧

## 目录
- 插件目录：`plugin/chrome_extension/mw_quick_archive_ext`

## 安装
1. 打开 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 复制mw_quick_archive_ext.zip到自己目录,解压
4. 选择解压目录：`mw_quick_archive_ext`

## 功能
- 配置后端 API 地址（默认：`http://127.0.0.1:8000`）
- 测试连接（请求 `GET /api/config`）
- 一键归档当前模型（请求 `POST /api/archive`）
- 同步 Cookie（自动读取）
- 手动设置 Cookie（扩展选项页）
- 打开本地模型主页（打开配置的 API 地址）
- 归档成功后可直接「打开本地模型」
- 模型页显示“已归档/未归档”状态与归档日期（YYYY-MM-DD）

## 使用流程
1. 点击扩展图标，先「保存地址」并「测试连接」
2. 按需点击右侧「标记开/标记关」控制页面状态标记显示
3. 打开 MakerWorld 模型页面（`https://makerworld.com.cn/zh/models/*`）
4. 点击「归档当前模型」
5. 若需要，点击「打开本地模型」查看归档页

## 手动 Cookie 设置
1. 扩展页点击「扩展程序详情」->「扩展程序选项」
2. 在「手动 Cookie」输入完整 Cookie 字符串
3. 点击「保存并同步手动 Cookie」
