# MakerWorld 油猴脚本使用说明

## 目录
- 脚本文件：`plugin/tampermonkey/mw_quick_archive.user.js`

## 安装
1. 浏览器安装 Tampermonkey
2. 新建脚本，粘贴 `mw_quick_archive.user.js` 内容并保存
3. 或直接拖拽该文件到 Tampermonkey 导入

## 功能
- 仅在模型页生效：`https://makerworld.com.cn/zh/models/*`
- 菜单配置后端 API 地址
- 菜单配置 API Token
- 手动设置并同步 Cookie
- 一键归档当前模型（右下角浮动按钮 + 菜单命令）

## 使用流程
1. 登录网页控制台，在“安全设置”页面生成一个 API Token
2. 打开任意 MakerWorld 模型页
3. Tampermonkey 菜单点击「⚙️ 设置后端地址与手动 Cookie」
4. 填写 API 地址、API Token、手动 Cookie，点击保存并同步
5. 点击页面右下角「📦 归档模型」即可归档
